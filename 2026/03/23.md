### ✅ 오늘 학습한 핵심 내용

#### <삼항 연산자>

> 조건 ? 참일 때 값 : 거짓일 때 값

- 항이 3개 : 조건, 참일 때 값, 거짓일 때 값
- ex) let result = isMyLayout ? myLayout : '';

---

#### <클래스>

> 1. class 본질 : 별명 붙이기

- HTML 태그한테 class="aaa"라고 하는 건,"너는 이제부터 'aaa'라는 별명을 가진 녀석이야!" 라고 선언하는 것이다.
- 얘를 부를 때 점(.)을 찍는다.

> 2.  Vue에서의 객체 모드 { }
>     > :class="{ '별명' : 조건 }"

- 조건이 true면 별명이 붙고, false면 별명이 떨어진다.

> 3.  Vue에서의 배열 모드 [ ]
>     > :class="[변수1, 변수2]"

- 기본 디자인 + 조건부 디자인
- ex) <div :class="{ 'red-text': hasError, 'bold-text': isImportant }"></div>

`HTML 태그에서 class를 쓸 경우에는 태그 내에서 먼저 정의하고, Vue의 경우에는 .클래스명 으로 스타일을 먼저 지정한다`

### 🔥 알게된 점

1. input 입력값을 숫자로 바꾸어줄 때,VM에서 숫자로 미리 변환해도 HTML 입력창을 거치면 글자로 변한다. 그래서 .number를 꼭 v-지시자에서 해주어야 한다.

2. v-on:으로 시작하는 모든 이벤트는 @로 바꿀 수 있다.

### ⚠️ 오답 노트

1. `v-on` 뒤에는 반드시 콜론(`:`)이 붙거나 `@`를 써야 한다. (예: `@click`)
2. 객체 내부 설정(`name`, `data`) 시에는 `=`이 아니라 `:`를 사용한다.
3. `v-model.number`를 쓰면 클릭 이벤트 로직에서 다음에 나오는 변수에 `parseInt()`를 생략할 수 있어 코드가 간결해진다.
