### ✅ 오늘 학습한 핵심 내용

#### <Props, Emit>

> 🌀컴포넌트 라이프사이클

Creation: 컴포넌트가 메모리에 만들어짐 (탄생)


Mounting: 우리 눈에 보이는 브라우저 화면에 딱 붙음 (출현)


Updating: 데이터(ref)가 변해서 화면이 다시 그려짐 (성장)


Unmounting: 화면에서 사라짐 (퇴장)

### 🔥 알게된 점

1. mounted란?

특징                    created (생성 완료)                       mounted (장착 완료)
 
데이터 접근           가능 (this.items 읽기/쓰기 가능)                      가능
화면(DOM) 접근        불가능 (아직 화면에 안 그려짐)                가능 (태그 조작 가능)
주요 작업,            서버 데이터 미리 요청 (Fetch API),           화면 조작,   차트 그리기, 포커스 주기"
