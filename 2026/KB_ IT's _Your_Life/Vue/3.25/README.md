#### <전역 지역 컴포넌트>

1. 지역 컴포넌트는 특정 파일 안에서만 쓸 수 있도록 직접 불러와 등록해 사용하는 전용 부품입니다. -> import + components 등록 필요

2. 전역 컴포넌트는 메인 설정에서 한 번만 등록하면 어디서든 불러오기 없이 바로 쓸 수 있는 공용 부품입니다. -> app.component('이름', 컴포넌트)

### 🔥 알게된 점

1. 부모-> 자식 : props(전달할 항복들을 지정, 항목=속성=특성)

2. 자식 -> 부모 : emit(자식컴포넌트에서 이벤트를 발생시키고 데이터를 부모컴포넌트로 전달),부모는 전달받은 이벤트의 결과를 처리할 메서드(함수)를 호출해서 원하는 처리를 한다.

3. Vite(비트)는 "초고속 엔진을 단 개발 도우미"
-> Vite('빠르다'는 뜻)는 거의 실시간으로 화면을 보여주는 최신 도구
