### ✅ 오늘 학습한 핵심 내용

#### <Props, Emit>
> 단방향 전달

- 부모 → 자식 (Props): 
    - 부모가 자식 태그에 v-bind 로 데이터 전달 
    -  자식은 props: [] 로 받기만 하며, 이 데이터는 읽기 전용  
    
- 자식 -> 부모 (Emit):
    -  자식이 값을 바꾸고 싶으면 $emit('이벤트명', 값)으로 신호 전달
    - 부모가 그 신호를 들어서(@이벤트명) 자기 데이터를 직접 수정 

---

#### <SFC (Single File Component)>

> 파일 하나(.vue)에 컴포넌트 하나를 통째로 담자.

 1. template : HTML 태그를 써서 블록의 뼈대를 잡습니다. 


2. script : JS로 데이터를 만들고 클릭 이벤트 같은 두뇌 역할을 넣습니다. 


3. style : CSS로 그 블록에만 적용될 옷을 입힙니다.
